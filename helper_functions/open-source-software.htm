Open Source Software

 

Cygwin (http://cygwin.com)

Cygwin is a large collection of GNU and Open Source tools (incl. the gcc/g++ compiler) which provide functionality similar to a Linux distribution for a PC running MS Windows. It includes a DLL that acts as an emulation layer providing substantial POSIX API functionality. This free software package is available at http://www.cygwin.com. The Cygwin DLL should work with all recent, commercially released x86 32-bit and 64-bit versions of MS Windows. In addition to the default packages, be sure to add the packages for gcc and make, so that you can run the C compiler. A helpful feature of Cygwin is that files (such as images) can be viewed via MS Windows applications by typing “cygstart <fname>” from a Cygwin shell.

 

In addition to the default packages, be sure to add the packages for gcc, gdb, make and patch. When running the Cygwin installation utility, there should be a “View” drop-down menu at the top left. From that menu, select “Category” to display the list of Cygwin packages sorted by category name. Next to the “Devel” category, click the “+” to expand the list of developer packages. Then toggle the “Skip” indicator next to the following packages so that they will be installed along with the default packages:

gcc-core: GNU Compiler Collection (C)

gcc-g++: GNU Compiler Collection (C++)

gdb: The GNU Debugger

patch: For applying patches

           

See https://cygwin.com/cygwin-ug-net.html for the Cygwin User’s Guide and the Quick Start Guide.

 

See https://cygwin.com/cygwin-ug-net/gdb.html for info regarding the gdb debugger, which allows you to set breakpoints and step through the execution of a program.

 

 

libpng (http://www.libpng.org/pub/png/libpng.html)

The libpng library is the official PNG reference library. It supports almost all PNG features and provides functions for manipulating (i.e., reading and writing) PNG image files. The libpng library is supported under Linux, MS Windows, Cygwin, MacOS, etc. It requires that the zlib library is already installed.

 

zlib (http://www.zlib.net)

The zlib library is designed to be a free, general-purpose, legally unencumbered -- that is, not covered by any patents -- lossless data-compression library for use on virtually any computer hardware and operating system. The zlib data format is itself portable across platforms. Unlike the LZW compression method used in the UNIX compress command and in the GIF image format, the compression method currently used in zlib essentially never expands the data. To install zlib under Cygwin, one must patch the zlib source. The patch is available at https://github.com/Alexpux/MSYS2-packages/raw/master/zlib/1.2.11-cygwin-no-widechar.patch.

 

 

 

Installation Notes

To install libpng and zlib in Cygwin, download the following files into the current directory: libpng-1.6.34.tar.gz and zlib-1.2.11.tar.gz and 1.2.11-cygwin-no-widechar.patch. Then do:

 

zcat zlib-1.2.11.tar.gz | tar xvf -

cd zlib-1.2.11

patch -p1 < ../1.2.11-cygwin-no-widechar.patch

./configure

make test

make install  # to install under /usr/local

# make install prefix=$HOME  # to install under $HOME

 

cd ..

 

zcat libpng-1.6.34.tar.gz | tar xvf -

cd libpng-1.6.34

./configure

make check

make install # to install under /usr/local

# make install prefix=$HOME  # to install under $HOME
