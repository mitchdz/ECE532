Computing Facilities and C/C++ Programming

 

ECE Compute Server (compute.engr.arizona.edu)

The College of Engineering maintains a “compute” Linux server for compiling programs and running computationally intense programs. For general questions regarding this server, contact ENGR IT support at support@engr.arizona.edu. To transfer files to and from the compute server, use a secure ftp application such as WinSCP. For login to the compute server, use a secure shell application such as PuTTY. See https://softwarelicense.arizona.edu/ssh-clients-windows-and-mac for further information about these and other open-source applications for sftp and ssh functionality for MS Windows and for Mac.

 

C/C++ Programming

There are many books and tutorials on C/C++ programming. For example, a first-time user may want to start with a very basic introduction to C, such as the tutorial at http://computer.howstuffworks.com/c.htm. The textbook used for ECE 175 Computer Programming is another example reference:

George Tselikis and Nikolaos Tselikas,

C: From Theory to Practice.

CRC Press, 2014.

 

The GNU C compiler (gcc) is one application for compiling a C program to obtain an executable file. The gcc compiler is already installed on the College of Engineering compute server and is also available on other systems running Linux, Cygwin, or MinGW. For further information on the gcc compiler, see https://gcc.gnu.org.
